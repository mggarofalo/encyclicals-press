# Vendored fonts

These fonts are vendored so the rendered PDF is consistent across machines
and across CI runs, regardless of what's installed system-wide.

## TeX Gyre Schola

A free Century Schoolbook clone maintained by GUST e-foundry.

* Files: `texgyreschola-{regular,italic,bold,bolditalic}.otf`
* Source: <https://www.gust.org.pl/projects/e-foundry/tex-gyre/schola>
* License: GUST Font License (GFL) — see <https://www.gust.org.pl/projects/e-foundry/licenses/GUST-FONT-LICENSE.txt>

## Inter

Used sparingly for folios. Designed by Rasmus Andersson.

* Files: `Inter-{Regular,Italic,Bold,BoldItalic}.otf`
* Source: <https://github.com/rsms/inter>
* License: SIL Open Font License 1.1
